# Tic-Tac-Toe Game

Предлагаю вашему вниманию игру *КРЕСТИКИ-НОЛИКИ* на основе библиотеки **tkinter**

* Запуск осуществляется через **gameApp.py**
* Реализована с помощью одного класса - **Game.py**
